# 📄 AI Resume Analyzer

A simple web application that analyzes a candidate's resume against a
job description and calculates a match score based on keyword
similarity.

This project demonstrates how recruitment tools can compare resumes with
job requirements.

------------------------------------------------------------------------

## 🚀 Features

-   Upload resume in **PDF format**
-   Paste a **Job Description**
-   Automatically extracts text from resume
-   Calculates **Match Score**
-   Displays:
    -   Matching Keywords
    -   Missing Keywords
    -   Visual Match Score

------------------------------------------------------------------------

## ⚙️ Installation

Clone the repository:

git clone https://github.com/yourusername/genai-resume-analyzer.git

Move into the project folder:

cd genai-resume-analyzer

Install dependencies:

pip install -r requirements.txt

------------------------------------------------------------------------

## ▶️ Run the Application

Run the Streamlit app:

python3 -m streamlit run app.py

Then open your browser and go to:

http://localhost:8501

------------------------------------------------------------------------

## 📊 Example Output

The application provides:

-   Match Score (percentage)
-   Matching keywords between resume and job description
-   Missing keywords from the resume
