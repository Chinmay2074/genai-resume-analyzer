# 🤖 AI Resume Analyzer

A Generative AI powered web application that analyzes a candidate's resume and compares it with a job description. The tool provides insights such as match score, key skills, missing skills, and improvement suggestions.

---

## 🚀 Features

* Upload Resume in **PDF format**
* Paste **Job Description**
* AI analyzes resume using **Google Gemini AI**
* Generates:

  * Match Score
  * Key Skills
  * Missing Skills
  * Improvement Suggestions

---

## 🛠 Tech Stack

* Python
* Streamlit
* Google Gemini AI
* PyPDF2

## ⚙️ Installation

Clone the repository

git clone https://github.com/Chinmay2074/genai-resume-analyzer?tab=readme-ov-file

Go to the project directory

cd genai-resume-analyzer

Install dependencies

pip install -r requirements.txt

---

## ▶️ Run the Application

streamlit run app.py

The application will open in your browser

http://localhost:8501

---

## 🔑 API Key Setup

This project uses **Google Gemini AI**.

Get your free API key from:

https://aistudio.google.com/app/apikey

Create API key and paste it in the application before analyzing the resume.
